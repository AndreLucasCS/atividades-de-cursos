Seletores personalizados

Um pouco sobre

Os seletores personalizados são uma forma de criar seletores que podem ser reutilizados em diferentes partes do seu código CSS. Eles permitem que você defina estilos específicos para elementos HTML sem precisar repetir o código várias vezes. Existem id e classes, que são os seletores mais comuns:

  1. Seletores de ID

    Os seletores de ID são usados para selecionar um único elemento HTML com um ID específico. Eles são precedidos pelo símbolo de hash (#) no CSS.

  2. Seletores de classe

    Os seletores de classe são usados para selecionar um ou mais elementos HTML com uma classe específica. Eles são precedidos pelo ponto (.) no CSS.

Objetivo de suas incrementação

Os seletores personalizados foram introduzidos no CSS para permitir que os desenvolvedores criem estilos reutilizáveis e organizados. Eles ajudam a manter o código CSS mais limpo e fácil de entender, além de facilitar a manutenção do estilo de um site.

A ideia é que você possa criar um seletor personalizado e usá-lo em diferentes partes do seu código, sem precisar repetir o mesmo estilo várias vezes. Isso economiza tempo e esforço, além de tornar o código mais legível.
