Formatações principais

Negrito / Destaque

Nesta frase temos uma parte em negrito criada a partir de uma tag não
semantica <b>.

Agora nesta frase há uma parte em destaque criada a partir da tag semântica
<strong>.

Itálico / Ênfase

Nesta frase, temos um termo em itálico usando a tag não semântica <i>.

Nesta frase, podemos observar uma parte em ênfase criada a partir da tag
semântica <em>.

Texto Mercado

Podemos criar também um texto marcado usando a tag <mark>, ela é usada
para marcar um texto muito valioso.

Texto grande e pequeno

Podemos também criar texto pequeno com a tag <small>, mas evite escrever
um texto grande com a tag obsoleta <big>, em vez disso, opite por usar
CSS.

Texto deletado

Essa é a tag <del> que cria um texto excluído, indicando que ele pode ser
lido, mas não considerado.

Texto inserido

Essa é a tag <ins> que faz o oposto da tag anterior, ela indica a inclusão de
um texto. Existe uma tag não semântica que deve ser evitada, ela faz algo
parecido com a tag <ins>, ela sublinha uma parte do texto. Ela é a tag <u>.

Texto sobrescrito

Use a tag <sup> para escrever coisas do tipo: 10-23 x 3,7378.

Texto subscrito

Use a tag <sub> para escrever coisas como: CO2.
